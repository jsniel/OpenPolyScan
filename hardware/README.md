# Hardware

Ce dossier contient les fichiers matériels du prototype OpenPolyScan.

## Contenu

- `cad/` : fichiers FreeCAD paramétriques
- `stl/` : exports STL imprimables

## Fonction

Ces fichiers permettent de fabriquer ou modifier le support mécanique du capteur
et les éléments imprimés associés au prototype.
