# Données

## Jeu de données principal

`raw/24_11_2025_PLA_PP_HDPE.csv`

## Description

Ce fichier contient des mesures spectrales acquises avec un capteur AS7265x.
La première colonne correspond au label matière.
Les colonnes suivantes correspondent aux bandes spectrales mesurées.

## Classes présentes

- PLA
- PP
- HDPE

## Format

- séparateur : `;`
- décimales : `,`

## Usage

Ce jeu de données sert de base d’expérimentation pour :
- l’entraînement de modèles simples
- l’évaluation comparative de classifieurs
- la documentation du protocole OpenPolyScan

## Limites

- taille encore modeste
- seulement trois familles de plastiques
- variabilité encore limitée des échantillons
- performances dépendantes du protocole d’acquisition
