# Contribuer à OpenPolyScan

Merci pour votre intérêt pour OpenPolyScan.

En contribuant à ce projet, vous acceptez que vos contributions soient publiées
sous les licences du dépôt :

- code : AGPL-3.0-or-later
- hardware : CERN-OHL-S-2.0
- documentation : CC BY-SA 4.0
- données : CC BY-SA 4.0, sauf mention contraire

## Contributions bienvenues

- amélioration du firmware ESP32
- ajout de scripts d’analyse ou d’entraînement
- amélioration des performances des modèles
- ajout de nouveaux jeux de données
- ajout de nouvelles classes de plastiques
- amélioration des fichiers CAO ou des supports imprimables
- amélioration de la documentation

## Recommandations

Merci de :
- documenter clairement toute modification importante
- conserver les crédits d’origine
- préciser l’origine et le protocole des nouvelles données
- indiquer les hypothèses et limites de toute nouvelle méthode proposée
